# Environment Variables

In order to run this project successfully you must supply the following environment variables:
***
**DATABASE_URL**
- example: **jdbc:mysql://localhost:3306/mydb**

***
**DATABASE_USERNAME**
- example: **root**

***
**DATABASE_PASSWORD**
- example: **mypassword**

***
**DATABASE_DRIVER**
- MySQL driver: **com.mysql.cj.jdbc.Driver**
